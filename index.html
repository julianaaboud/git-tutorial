Git é um sistema de controle de versão distribuído e um sistema de gerenciamento de código fonte, com ênfase em velocidade. O Git foi inicialmente projetado e desenvolvido por Linus Torvalds para o desenvolvimento do kernel Linux, mas foi adotado por muitos outros projetos.

Comandos

Comando Descrição
git config --global user.name "nome do usuario" Configura a identificação do nome de quem está utilizando o git
git config --global user.email "email@x.com.br" Configura a identificação do e-mail de quem está utilizando o git
git init Starta o monitoramento do git em determinada pasta
git status Mostra o que foi modificado / adicionado / deletado da pasta
git remote add origin https://github.com/... Adiciona a origem do repositório
git add nome-do-arquivo Adiciona arquivos em específico
git add . Adiciona todos os arquivos/diretórios
git commit -m "descrição" Adiciona uma tag para subir o arquivo
git push origin master Adiciona a configuração para o github
git clone https://githu.com/ Baixa o repositório para trabalhar na máquina
git checkout -b git Cria uma branch
